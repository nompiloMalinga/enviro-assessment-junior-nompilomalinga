# Enviro365 Investments 

A full-stack application built for Enviro365 Investments to automate the withdrawal notice process, allowing investors to view portfolios, submit withdrawals, and download CSV statements.

## Backend
Built with Spring Boot, Spring Data JPA, and H2 database.

See [backend README](./backend/README.md) for full details on endpoints, business rules, and setup instructions.

## Frontend
Built with Angular and Angular Material.

See [frontend README](./frontend/README.md) for full details on features, validation, and setup instructions.
